# 📊 Tabela de Gastos Pessoais

Uma solução desktop robusta e intuitiva para o **gerenciamento financeiro doméstico**. Este projeto foi desenvolvido para facilitar o controle de despesas mensais, permitindo uma visão clara de gastos fixos, parcelamentos e status de pagamento.

---

## 🎯 Objetivo do Projeto
Transformar a organização financeira em algo simples e digital. A aplicação substitui planilhas complexas por uma interface focada na experiência do usuário, garantindo que o controle de contas não seja uma tarefa árdua.

## 🚀 Funcionalidades Principais
* **Gestão de Contas:** Cadastro detalhado com nome da conta, valor, mês/ano e data de vencimento.
* **Controle de Parcelamentos:** Opção de marcar contas parceladas com identificação do número da parcela.
* **Filtros Dinâmicos:** Pesquisa rápida por mês ou status de pagamento (Pago/Pendente).
* **Cálculos Automáticos:** Somatório total de gastos visível em tempo real conforme os filtros aplicados.
* **Integração com Excel:** Exportação e importação de dados para arquivos `.xlsx`, permitindo backups e análises externas.

---

## 🛠️ Tecnologias Utilizadas
O projeto utiliza tecnologias consolidadas do ecossistema Java para garantir estabilidade e performance:
* **Java (JDK 11+):** Linguagem principal.
* **Java Swing:** Para a criação da Interface Gráfica (GUI).
* **Apache POI:** Biblioteca para manipulação profissional de arquivos Excel.
* **Programação Orientada a Objetos (POO):** Estrutura de classes clara e reutilizável.
* **Java Streams:** Para filtragem e processamento eficiente de dados.
